**Tags:** one-column, portfolio, custom-background, footer-widgets, full-width-template, translation-ready, blog, news

FullWidther is a full width WordPress theme mainly created for landing pages and blogs. It's a elegant, clean and multipurpose theme where you can customize colors to make the theme dark, light or colorful. We have lots of color options so you can make the theme personal. Full Widther is responsive and built on HTML5, CSS3, it is also fully SEO optimized. The theme is also suitable for photography and portfolios, especially writer portfolios since the typography is optimized for writing. Full Widther works with most page builders so you can create a creative landing page or one pager for your liking.

## Description

FullWidther is a full width WordPress theme mainly created for landing pages and blogs. It's a elegant, clean and multipurpose theme where you can customize colors to make the theme dark, light or colorful. We have lots of color options so you can make the theme personal. Full Widther is responsive and built on HTML5, CSS3, it is also fully SEO optimized. The theme is also suitable for photography and portfolios, especially writer portfolios since the typography is optimized for writing. Full Widther works with most page builders so you can create a creative landing page or one pager for your liking.
